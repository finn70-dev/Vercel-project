# Xylanthen

A modern, lightweight platform for processing data and generating interactive charts.

## Key Features
* **Visuals:** Fast chart generation (Line, Bar, Pie, Radar) via Chart.js.
* **Analytics:** Automatic stats (sum, average, median, min/max).
* **Tech:** Built with Node.js serverless functions and Vercel.

## Quick Start
1. Clone the repo: `git clone <your-repo-url>`
2. Install: `npm install`
3. Run: `npm run dev`
